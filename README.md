# 카카오톡 클론코딩 프론트엔드

## 수정 내역

### 기능 추가/수정
---
로그인 실패시 문구 ID/PW 확인 문구 출력 추가

로그인창 로고 이미지 직접 불러오는 걸로 수정함

회원가입 폼 새로 작성하였음

친구추가 관련 오류처리 추가(본인을 친구추가하는 경우, 없는 사용자를 친구추가하는 경우)

알림메시지 사라짐 로직 추가

채팅방 가운데 정렬

상태메시지 불러오지 못하는 오류 해결

상태메시지 변경 기능 추가

날짜 동기화

읽음 기능 추가

안 읽은 메시지 수 표시

채팅방에서 상대방 이름 보이게 함

채팅방 내부에 있어도 메시지 읽음 표시 업데이트되도록 함

### 오류 수정
---
프로필 클릭 오류 해결)
이전: 내 프로필 클릭시 내 프로필이 아닌 최근에 클릭한 친구의 프로필이 나오는 오류 해결

채팅방에서 나와도 메시지를 읽지 않았다고 뜨는 오류 해결
 
### 기타
---
tsconfig.json에서 suppressImplicitAnyIndexErrors 옵션은 더이상 쓰이지 않아 제거함

채팅방에서 나왔을 때 속하지 않은 채팅방이 뜨는 오류

- 새로고침, 코드수정 등 강제로 로그인 화면에 진입시 이전 로그인 유저의 채팅방 목록이 남음.
로그아웃 기능으로 로그아웃 후 로그인시 정상동작함을 확인
